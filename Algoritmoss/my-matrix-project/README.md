# My Matrix Project

Este proyecto crea matrices y las guarda en archivos .txt.

## Cómo usar

1. Asegúrate de tener Node.js instalado en tu máquina.

2. Clona este repositorio a tu máquina local.

3. Navega a la carpeta del proyecto y ejecuta `npm install` para instalar las dependencias.

4. Ejecuta `node src/createMatrix.js` para crear una matriz y guardarla en un archivo .txt en la carpeta de salida.

## Estructura del proyecto

- `src/createMatrix.js`: Este es el punto de entrada de la aplicación. Crea una matriz y la guarda en un archivo .txt.

- `output`: Esta carpeta almacenará los archivos .txt generados por el script createMatrix.js.

- `package.json`: Este archivo es el archivo de configuración para npm. Enumera las dependencias y scripts para el proyecto.

Este proyecto no tiene dependencias externas.
